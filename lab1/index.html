Шифрование и дешифрование текста

Улучшенный столбцовый метод перестановок. Ключ — слово из английских букв. Строки таблицы имеют переменную длину: строка i заполняется до столбца с номером i. Шифруются только английские буквы (A-Z), остальные символы игнорируются.
Шифр Виженера с самогенерирующимся ключом (автоключ по открытому тексту). Ключ — слово из русских букв. Алфавит: А-Я (33 буквы, с Ё и Й). Индексация с 1. Шифруются только русские буквы, остальные символы игнорируются.
